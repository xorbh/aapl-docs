# WI-007: Assembly & Component Testing

**Document ID**: WI-007
**Effective Date**: 2024-11-26
**Last Revised**: 2024-11-26
**Author**: Manufacturing Engineering
**Approval**: Production Manager

---

## Purpose & Scope

This work instruction provides procedures for assembling components into sub-assemblies and final assemblies, and performing required functional testing. Assembly operations must follow detailed specifications to ensure proper fit, function, and quality.

**Operations Covered**:
- Component assembly
- Sub-assembly procedures
- Final assembly verification
- Functional testing
- Performance validation

**Personnel**: Assemblers, technicians, quality inspectors

---

## Before You Start

### Required Training & Certification
- [ ] Completed Assembly Training
- [ ] Familiar with engineering drawings
- [ ] Trained in precision measurement
- [ ] Completed Testing Equipment Training (if applicable)

### Required Materials & Tools

**Assembly Components**:
- All components per bill of materials (BOM)
- Fasteners (bolts, screws, pins, clips)
- Adhesives or sealants (if applicable)
- Shims or spacers

**Tools**:
- Screwdrivers (multiple sizes)
- Wrenches and socket sets
- Precision measurement tools
- Torque wrench (for critical fasteners)
- Snap ring tools (if applicable)
- Assembly fixtures or jigs
- Work platform or assembly table

## Step 1: Assembly Preparation

### Component Verification

1. **Gather all components**:
   - Review bill of materials (BOM)
   - Verify all components present
   - Confirm part numbers match specification
   - Check component quantity matches BOM

2. **Inspect components before assembly**:
   - [ ] No damage, cracks, or defects
   - [ ] Surfaces clean and dry
   - [ ] No rust or corrosion
   - [ ] All required features present
   - [ ] Fastener quality acceptable
   - Record inspector initials on assembly record

3. **Organize components**:
   - Arrange in assembly sequence order
   - Place in easy-access locations
   - Group fasteners and hardware
   - Verify correct materials for assembly:
     - Lubricants (if required)
     - Adhesives (if required)
     - Sealants (if required)

### Assembly Documentation Review

1. Review engineering drawing:
   - Identify all assembly steps
   - Note critical dimensions/tolerances
   - Identify required fastener types
   - Note torque specifications
   - Identify sequence or special procedures

2. Review work order:
   - Quantity to assemble
   - Assembly deadline
   - Specific customer requirements
   - QA hold points (if any)

3. Prepare assembly fixtures:
   - Assemble any required fixtures or jigs
   - Verify fixtures secure parts properly
   - Test fixtures with first assembly unit

---

## Step 2: Component-to-Component Assembly

### Basic Assembly Steps

1. **Position first component**:
   - Place base component on work platform
   - Verify stable and secure
   - Verify correct orientation per drawing

2. **Position second component**:
   - Align with first component per drawing
   - Verify alignment with straightedge or gage
   - Check that features align properly:
     - Holes line up
     - Edges are flush
     - Surfaces contact fully

3. **Install fasteners**:
   - For bolted joints:
     - Insert bolt through aligned holes
     - Install washers if specified
     - Install nut hand-tight
     - Verify no cross-threading
   - For threaded inserts:
     - Check insert is secure in base part
     - Insert screw
     - Turn gently to seat—do not force
   - For rivets or permanent fasteners:
     - Follow special procedure (see below)

4. **Tighten fasteners**:
   - For critical fasteners (specified on drawing):
     - Use torque wrench
     - Apply torque specification from drawing
     - Verify torque reading on wrench
     - Record torque applied
   - For non-critical fasteners:
     - Tighten firmly with hand tools
     - Verify secure (no movement)
     - Do not over-tighten

### Fastener Installation Details

**Bolt & Screw Installation**:
1. Verify fastener type matches specification:
   - Correct diameter (e.g., 1/4", 5/16")
   - Correct length
   - Correct material (steel, stainless, etc.)
   - Correct drive type (Phillips, hex, etc.)

2. Check fastener condition:
   - No stripped threads
   - No rust or corrosion
   - Head not damaged

3. Installation sequence:
   - Insert fastener
   - Tighten diagonal opposite fasteners (if multiple):
     - Prevents uneven clamping
     - Prevents part warping
     - Example for 4 bolts: tighten in pattern 1-3-2-4
   - Verify no gaps between parts after tightening

**Adhesive or Sealant Application**:
1. If adhesive required:
   - Verify adhesive type matches specification
   - Check expiration date
   - Apply adhesive per manufacturer instructions:
     - Thin, even bead
     - Along entire joint if required
     - Avoid excess (prevents squeeze-out)
   - Allow cure time per specification before assembly stress
   - Follow safety procedures (ventilation, PPE)

2. If sealant required:
   - Apply to contact surfaces
   - Controlled amount to avoid mess
   - Allow cure time (do not handle until set)

### Snap Ring or Retaining Ring Installation

1. For snap rings on shafts:
   - Place snap ring tool on ring
   - Compress ring slightly
   - Slide onto shaft to correct groove
   - Release tool—ring should seat firmly
   - Verify ring is fully seated

2. For snap rings in bores:
   - Place snap ring tool on ring
   - Expand ring
   - Position in bore to correct groove
   - Release tool—ring should seat firmly
   - Verify ring is fully seated

## Step 3: Sub-Assembly Verification

### Dimensional Verification

1. After each major assembly step:
   - Stop and verify critical dimensions
   - Use precision measurement tools
   - Compare to drawing specification
   - If dimension out of tolerance, correct before continuing

2. Common critical dimensions to check:
   - Overall length/width/height
   - Center distances between components
   - Alignment of features
   - Stack-up of component tolerances

### Mechanical Verification

1. Check assembly function:
   - If movable parts: verify smooth motion
   - If mechanical advantage: verify proper operation
   - If alignment critical: verify parts aligned
   - If locked position required: verify security

2. Test fit with mating components:
   - If assembling sub-assembly that mates to other parts
   - Try fit with mating components (if available)
   - Verify no interference
   - Verify proper alignment

---

## Step 4: Final Assembly

### Assembly of Sub-Assemblies

1. Position first sub-assembly:
   - Place on work platform
   - Secure if required
   - Verify correct orientation

2. Position additional sub-assemblies:
   - Align per drawing
   - Install fasteners
   - Apply torque specifications
   - Verify alignment after each sub-assembly

3. Complete final assembly:
   - Install any remaining components
   - Verify all components present
   - Verify all fasteners tight
   - Verify nothing missing

### Final Assembly Documentation

1. Record assembly completion:
   - Verify all components present (check off BOM)
   - Verify all fasteners installed
   - Note any substitutions or deviations
   - Record assembler initials

2. Identify assembly for next step:
   - Label with work order number
   - Label with assembly date
   - Flag for testing (if required)

## Step 5: Functional & Performance Testing

### Test Preparation

1. Set up testing station:
   - Gather test equipment
   - Verify equipment is calibrated
   - Set up test fixtures
   - Allow equipment to warm up (if electrical)

2. Verify test procedure:
   - Confirm test specification from work order
   - Review test procedure documentation
   - Prepare test data sheet
   - Verify safety procedures

### Common Tests

**Rotation/Movement Test**:
1. If assembly has rotating or moving parts:
   - Manually rotate or move assembly
   - Verify smooth operation
   - Listen for grinding, binding, or noise
   - Feel for resistance or binding
   - Record: PASS / FAIL

**Load Test**:
1. If assembly must support load:
   - Apply specified load
   - Observe for deflection or movement
   - Check for stress points
   - Verify assembly doesn't fail
   - Record: PASS / FAIL / Load value

**Pressure/Seal Test**:
1. If assembly must hold pressure or be sealed:
   - Apply test pressure per specification
   - Monitor pressure over time (check for leaks)
   - Inspect for leaks or seepage
   - Pressurize for specified duration
   - Record: PASS (no leaks) / FAIL (leak detected)

**Electrical Test** (if applicable):
1. If assembly has electrical components:
   - Verify proper connections
   - Check continuity with multimeter
   - Verify proper voltage levels (if powered)
   - Check for shorts or faults
   - Record: PASS / FAIL

**Performance Test**:
1. If assembly has functional purpose:
   - Operate assembly under normal use conditions
   - Measure performance against specification:
     - Output (speed, force, power, etc.)
     - Efficiency
     - Safety limits
   - Compare to specification
   - Record: PASS / FAIL / Actual value

### Test Data Recording

1. Complete test sheet:
   - Date and time of test
   - Assembler/technician name
   - Assembly/work order number
   - Test type
   - Test parameters (load, pressure, speed, etc.)
   - Results (pass/fail)
   - Actual measured values
   - Any observations or issues
   - Signature and date

2. If test fails:
   - STOP assembly process
   - Notify supervisor
   - Diagnose failure reason
   - Determine if rework needed
   - Document failure and corrective action

## Step 6: Assembly Rework (if required)

### Addressing Failed Tests or Damage

1. **Disassembly for rework**:
   - Carefully remove fasteners
   - Separate components gently
   - Avoid damaging components
   - Inspect components for damage
   - Preserve any component needing replacement

2. **Identify rework requirement**:
   - Determine which component caused failure
   - Verify component specification
   - Order replacement if needed
   - Repair if possible

3. **Reassembly after rework**:
   - Follow assembly procedure again
   - Use replacement component
   - Re-test to verify pass

---

## Step 7: Cleaning & Finishing

### Assembly Cleanup

1. Remove protective coverings:
   - Remove plastic protective films
   - Remove assembly tape or labels
   - Clean away any assembly debris

2. Clean assembled product:
   - Wipe with clean, dry cloth
   - Remove dust and oils
   - For precision surfaces: use lens-safe cloth
   - Verify clean and presentable

### Protective Treatment (if required)

1. If corrosion protection required:
   - Apply protective coating per specification
   - Follow drying time requirements
   - Do not handle until dry

2. If lubrication required:
   - Apply specified lubricant per specification
   - Light coat—avoid excess
   - Allow penetration time

3. If final assembly sealing required:
   - Apply sealant per specification
   - Observe required cure time

---

## Step 8: Quality Inspection & Final Documentation

### Final Inspection

1. Visual inspection:
   - Overall appearance acceptable
   - No visible damage or defects
   - All components present and secure
   - Fasteners all tight and secure

2. Dimensional verification (critical dimensions):
   - Measure key dimensions with appropriate tools
   - Verify within tolerance
   - Compare to original specification

3. Functional verification:
   - Quick operational test
   - Verify smooth operation
   - No unusual sounds or behavior

### Completion Documentation

1. Complete assembly work order:
   - All assembly steps completed
   - All tests passed
   - Quantity assembled verified
   - Quality sign-off
   - Assembler and inspector signatures

2. Prepare for next step:
   - Label with "READY FOR SHIPMENT" or next process
   - Move to staging area
   - Update work order status in system
   - File assembly documentation

## Safety Reminders

- **Ergonomics**: Maintain proper posture, avoid repetitive strain
- **Sharp edges**: Watch for sharp edges on machined components
- **Heavy assemblies**: Use lift equipment for heavy assemblies (>30 lbs)
- **Tool safety**: Use appropriate tools and handle safely
- **Chemical safety**: Follow safety procedures for adhesives/sealants (ventilation, PPE)
- **Electrical**: Never work on live electrical assemblies

## Troubleshooting

### Problem: Components Don't Align
- [ ] Verify correct components (check part numbers)
- [ ] Check component dimensions (may be out of spec)
- [ ] Use assembly fixture to force correct alignment
- [ ] Contact engineering if consistent misalignment

### Problem: Fastener Won't Tighten
- [ ] Check for stripped threads
- [ ] Verify correct fastener type
- [ ] Back off and re-insert slowly
- [ ] If cross-threaded, remove and start again
- [ ] Discard damaged fastener

### Problem: Test Fails Repeatedly
- [ ] Verify test equipment is calibrated
- [ ] Verify test procedure correct
- [ ] Review assembly procedure—may have been error
- [ ] Consider component defect
- [ ] Notify supervisor and quality for investigation
