|---|---|---|

### Daily Maintenance Checklist
- [ ] Spindle speed verification (no load, RPM gauge check)
- [ ] Coolant level check (should be at fill line)
- [ ] Visual inspection for leaks
- [ ] Machine sound normal (no grinding or squealing)
- [ ] Emergency stop button test
- [ ] Chuck security test (no slippage)

### Weekly Maintenance
- [ ] Change coolant filter
- [ ] Wipe down spindle area
- [ ] Lubricate turret/tool changer
- [ ] Check hydraulic pressure
- [ ] Verify all light indicators

### Monthly Maintenance
- [ ] Tool offset calibration
- [ ] Spindle runout check (<0.002")
- [ ] Ball screw lubrication
- [ ] Coupling inspection
- [ ] Electrical connector inspection

### Quarterly Maintenance
- [ ] Full spindle inspection (by technician)
- [ ] Coolant system flush and refill
- [ ] Belt/chain condition assessment
- [ ] Bearing temperature monitoring
- [ ] Complete system diagnostics

### Common Parameters

| Operation | Spindle Speed | Feed Rate | Depth of Cut | Coolant |
|---|---|---|---|---|
| Aluminum roughing | 2,000-3,000 RPM | 10-20 IPM | 0.100"-0.200" | Flood |
| Aluminum finishing | 4,000-5,000 RPM | 8-15 IPM | 0.025"-0.050" | Flood |
| Steel roughing | 800-1,200 RPM | 5-10 IPM | 0.050"-0.100" | Flood |
| Steel finishing | 1,500-2,500 RPM | 5-8 IPM | 0.010"-0.025" | Flood |
| Stainless roughing | 600-900 RPM | 3-6 IPM | 0.050"-0.100" | Flood |
| Stainless finishing | 1,000-1,500 RPM | 3-5 IPM | 0.010"-0.025" | Flood |

### Troubleshooting Quick Guide

| Problem | Symptoms | Check | Solution |
|---|---|---|---|
| **Chatter marks** | Surface ripples | Spindle speed | Reduce speed by 20% |
| **Tool breakage** | Tool fractures | Feed rate, depth | Reduce feed or DOC |
| **Coolant not flowing** | Dry chips | Pump, nozzles | Check filter, unclog nozzles |
| **Rough surface** | Poor finish | Tool wear, speed | Replace tool or increase speed |
| **Dimension drift** | Out of tolerance | Tool offset, thermal | Reset offset, cool spindle |
| **Spindle noise** | Grinding sound | Bearing, spindle | Call maintenance |

### Spare Parts (Common)
- End mills (1/2", 3/8", 1/4") - carbide
- Face mills (2" diameter)
- Drills (fractional sizes 1/16" to 1")
- Collets (ER32 set)
- Chuck wrench
- Tool change parts (wear items)
- Spindle belt
- Coolant filter cartridges

### Grinding Wheel Specifications

### Daily Maintenance
- [ ] Wheel balance verification (no vibration)
- [ ] Coolant level and color check
- [ ] Wheel surface inspection for cracks
- [ ] Work table cleanliness
- [ ] Coolant nozzle position check
- [ ] Emergency stop test

### Weekly Maintenance
- [ ] Wheel dressing (maintain sharpness)
- [ ] Coolant filter change
- [ ] Magnetic chuck cleaning
- [ ] Spindle lubrication check

### Monthly Maintenance
- [ ] Complete wheel inspection
- [ ] Coolant system analysis
- [ ] Bearing inspection
- [ ] Electrical connection check

### Quarterly Maintenance
- [ ] Full system diagnostic
- [ ] Spindle replacement assessment
- [ ] Coolant complete change
- [ ] Seal inspection/replacement

### Grinding Parameters by Material

| Material | Wheel Speed | Table Feed | Depth of Cut | Coolant |
|---|---|---|---|---|
| Steel | 5,500 SFM | Slow (0.5") | 0.001"-0.002" | Flood |
| Stainless | 5,500 SFM | Very slow (0.25") | 0.0005"-0.001" | Flood |
| Aluminum | 4,500 SFM | Slow (0.5") | 0.001"-0.002" | Flood |
| Cast Iron | 5,500 SFM | Medium (1") | 0.002"-0.003" | Flood |
| Carbide | 6,500 SFM | Very slow | 0.0005" | Flood |

### Troubleshooting

| Problem | Cause | Solution |
|---|---|---|
| **Rough finish** | Glazed wheel | Dress wheel |
| **Chatter** | Loose work piece | Re-clamp |
| **Overheating** | Insufficient coolant | Check flow/concentration |
| **Wheel chipping** | Spindle speed too high | Reduce speed |
| **Poor flatness** | Worn chuck | Contact maintenance |

### Spare Parts
- Grinding wheels (120 grit standard)
- Diamond dresser
- Coolant filters
- Spindle brushes (if brush-type)
- Coolant (25 gallons)
- Work holding magnets

### Daily Checks
- [ ] Power on/off test
- [ ] Torch/electrode holder condition
- [ ] Cable insulation check
- [ ] Gas flow verification
- [ ] Cooling water flow (TIG)
- [ ] Ground connection security

### Weekly Maintenance
- [ ] Torch tip cleaning/replacement
- [ ] Contact tip inspection (MIG)
- [ ] Water cooler filter check (TIG)
- [ ] Hose inspection for damage
- [ ] Electrical connector cleaning

### Monthly Maintenance
- [ ] Complete torch inspection
- [ ] Welding machine diagnostics
- [ ] Cooling system flush (TIG)
- [ ] Gas regulator servicing

### Spare Parts
- Contact tips (MIG)
- Nozzles (MIG)
- Tungsten electrodes (TIG)
- Filler rods (ER70S, ER308L, ER4043)
- Welding cable (25 ft)
- Gas hose (50 ft)

### Safety Checks
- [ ] Shielding gas present (check bottle pressure)
- [ ] Gas flow meter set correctly
- [ ] Ground clamp firmly attached
- [ ] No water leaks (TIG)
- [ ] Proper ventilation active
