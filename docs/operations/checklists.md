# Operational Checklists

This document provides standardized checklists for key manufacturing operations. Use these checklists to ensure consistency, quality, and safety across all shifts and departments.

## Pre-Shift Equipment Checklist

### Daily Equipment Safety Check

**Equipment ID**: _____________ **Date**: _____________ **Operator**: _____________

```mermaid
checklist
    title Daily Pre-Operation Equipment Check
    - [ ] Visual inspection: No visible damage or unusual wear
    - [ ] Safety guards present and properly positioned
    - [ ] All emergency stops functional and accessible
    - [ ] Coolant/lubrication levels adequate
    - [ ] Compressed air lines secured with no leaks
    - [ ] Electrical connections secure and no exposed wires
    - [ ] Tool holders installed and secured properly
    - [ ] Workspace clear of obstructions and debris
    - [ ] Machine sound normal, no unusual noises
    - [ ] All displays/gauges responding properly
    - [ ] No active warning lights or alarms
    - [ ] Floor around equipment dry and clean
```

**Pre-Startup Test Run**:
- [ ] Equipment powers on properly
- [ ] All controls respond correctly
- [ ] Movement is smooth and normal
- [ ] No vibration or excessive noise
- [ ] Temperature readings normal
- [ ] Pressure gauges within normal range

**Sign-Off**: _________________________ **Time**: _________ **Status**: ☐ PASS ☐ FAIL

**If FAIL, notify supervisor immediately. Do not operate equipment.**

## Chemical Spill Response Checklist

### Spill Incident Protocol

**Location**: _________________________ **Date**: _____________ **Time**: _________

**Chemical(s) Involved**: _________________________________________________________________

**Estimated Quantity**: _________________________ **Reported By**: _________________________

```mermaid
checklist
    title Chemical Spill Response
    - [ ] Area immediately cleared of personnel
    - [ ] Supervisor notified immediately
    - [ ] Spill size assessed less than 5 gallons
    - [ ] All ignition sources identified and controlled
    - [ ] Spill kit retrieved and personnel briefed
    - [ ] PPE donned properly gloves, coveralls
    - [ ] Absorbent material applied to spill
    - [ ] Spill completely contained
    - [ ] Contaminated materials placed in proper waste container
    - [ ] Area decontaminated with appropriate cleaner
    - [ ] SDS reviewed for hazard information
    - [ ] Incident documented and reported
    - [ ] Area cordoned off until dry
```

**For Large Spills (>5 gallons)**:
- [ ] Area evacuated completely
- [ ] Emergency services called 911
- [ ] All personnel moved to safe distance
- [ ] Emergency services notified of chemical type
- [ ] Wait for professional cleanup
- [ ] Document incident thoroughly

**Spill Size**: ☐ SMALL <5 gal ☐ LARGE >5 gal

**Trained Personnel Present**: ☐ YES ☐ NO

**Incident Report Filed**: ☐ YES ☐ NO **Report #**: _____________
